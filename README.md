# hndy-factorio-color

## Overview

A utility library for *Factorio* mods providing classes and functions for working with various color spaces and color values.

## Color Spaces

- RGB (red, green, blue)
- sRGB (gamma-corrected RGB)
- HSV (hue, saturation, value)
- HSL (hue, saturation, lightness)
- HWB (hue, whiteness, blackness)
- CIELAB (lightness, green-to-red, blue-to-yellow)
- CIELCh (lightness, chroma, hue)
- Oklab (lightness, green-to-red, blue-to-yellow)
- Oklch (lightness, chroma, hue)

## Operations

- Conversion between color spaces
- Conversion to and from game color
- Conversion to and from CSS components
- Gamut clamping
- Clone with mutation
- Interpolation
